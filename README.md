# ✨ Overview

Harmonic is a modern open-source music streaming platform designed to provide a beautiful listening experience with a clean interface, lightning-fast performance, and powerful music management features.

<br>

<div align="center">

| 🎵 Streaming | 🔍 Discovery | 📂 Playlists |
|-------------|-------------|-------------|
| High-quality audio streaming | Search artists, albums & tracks | Create and manage playlists |

| ❤️ Library | 🌙 Dark Mode | 📱 Responsive |
|------------|-------------|-------------|
| Save favorite songs | Beautiful modern UI | Works across devices |

</div>
